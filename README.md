# AnorMotion

## Telegram Mini App with Video Support

This project is a Telegram Mini App built with React that includes a video player designed for mobile viewports.

### Project Structure
- `MainBot.js`: Telegram bot logic based on `node-telegram-bot-api`.
- `src/App.js`: React frontend for the Mini App.
- `public/index.html`: Includes the Telegram Web App SDK.
- `src/App.css`: Premium dark mode styling.

### Getting Started

1.  **Bot**: Run `node MainBot.js` (make sure `.env` contains your `BOT_TOKEN`).
2.  **Web App**: Run `npm start` to see the local version.
3.  **Deploy**: Deploy to Vercel/Netlify for a public HTTPS URL.
4.  **Connect**: Update the URL in `MainBot.js` with your deployed URL.
